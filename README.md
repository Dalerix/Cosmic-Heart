# Cosmic Love Heart

Cinematic interactive WebGL heart made from animated particles.

## Project Structure

```text
cosmic-love-heart/
├── index.html
├── src/
│   ├── main.js
│   └── styles.css
└── README.md
```

## How To Run

Open `index.html` directly in a browser.

No installation is required. The project uses plain WebGL, CSS, and JavaScript, so it works well for GitHub Pages.

## GitHub Pages

1. Push this folder to GitHub.
2. Open repository settings.
3. Go to `Pages`.
4. Select the `main` branch and `/root`.
5. Save, then open the generated GitHub Pages link.

## Controls

- Move mouse or finger to distort the particles.
- Click or tap to explode the heart, reveal wing trails, and reform it.
